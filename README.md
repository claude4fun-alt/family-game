# 🎮 המשחק המשפחתי - משחק תחנות QR

מערכת משחק משפחתי אינטראקטיבי: 10 תחנות עם קודי QR, שאלות טקסט/תמונה, ניקוד וזריזות, ולוח תוצאות חי עם המחשת בניינים.

## הפעלה מקומית
\`\`\`
npm install
npm start
\`\`\`
פתח http://localhost:3007

## העלאה ל-Render (חינמי)
1. העלה את התיקייה ל-GitHub.
2. ב-render.com → New → Web Service → חבר את ה-repo.
3. Build: \`npm install\` | Start: \`npm start\`
4. הוסף Environment Variable: \`ADMIN_PASS\` = הסיסמה שלך.
5. אחרי הפריסה, קבל כתובת כמו \`https://family-game.onrender.com\`.
6. בדשבורד → טאב QR → הדבק את הכתובת ב"כתובת בסיס" → צור הכל → הדפס.
